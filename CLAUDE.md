## Язык и стиль общения

Все комментарии, документация и описания в проекте пишутся на **русском языке**.
При общении с пользователем использовать **мужской род** (например: "я сделал", "я добавил").

## Обзор проекта

Университетский проект: Go REST API бэкенд, который оценивает вероятную оценку студента по предмету на основе входных параметров. Фронтенд на Vue.js.

## Стек

- **Бэкенд**: Go (REST API)
- **Фронтенд**: Vue.js
- **БД**: PostgreSQL
- **Инфраструктура**: Docker (docker-compose), локальный запуск
